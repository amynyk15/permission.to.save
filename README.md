# Permission to Save Web App

一個純 HTML / CSS / JavaScript 的 Budget Planner + Shopping List 小網站。

## 如何使用

1. 在 VS Code 建立資料夾，例如 `permission-to-save-webapp`
2. 放入以下三個檔案：
   - `index.html`
   - `style.css`
   - `script.js`
3. 直接用瀏覽器打開 `index.html`
4. 建議在 VS Code 安裝 Live Server，右鍵 `index.html` → Open with Live Server

## 功能

- Monthly Dashboard
- Budget Planner
- Actual Expenses
- Saving Goals
- Shopping List
- Shopping item 可一鍵加入 Actual Expenses
- 資料會儲存在瀏覽器 localStorage
